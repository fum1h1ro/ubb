# Ubb

Helper for Unity batch build.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ubb'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ubb

## Usage

```
$ ubb export -o hoge.unitypackage Plugins/hoge
$ ubb import hoge.unitypackage
```
